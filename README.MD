# HAR Extractor

1. Na primeira execução, pede a pasta do projeto PLL Core Platform e guarda esse caminho em config.json.
2. Detecta a branch atual do projeto.
3. Pede o nome do ficheiro .har que está na pasta do script.
4. Extrai as entradas para uma pasta com o mesmo nome do .har.

Como usar:
- Coloca o ficheiro .har na pasta do script.
- Executa: python main.py
